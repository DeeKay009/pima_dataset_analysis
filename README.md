# Pima Diabetes Analysis
This project analyzes the Pima Indians Diabetes dataset using Python and machine learning.

## Steps:
- Data cleaning
- Feature preparation
- Model building (Logistic Regression)
- Evaluation and improvement
# the process 
The goal was to predict whether a patient has diabetes based on clinical features. I cleaned the data using median imputation for invalid zero values and performed scaling to normalize feature ranges. I also performed exploratory data analysis  to identify patterns, distributions, correlations, and potential outliers in variables such as glucose, BMI, and age. I built a Logistic Regression model( as the target variable- outcome was binary) and evaluated it using accuracy, precision, recall, F1-score, and ROC-AUC. especially recall. The reason eas it is a moderately imbalanced dataset and two, the goal is to detect diabteoc cases. The final outcome was a moderately accurate model that can help in early identification of diabetes risk, with glucose and BMI as key predictors.
